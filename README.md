Proyecto de páginas y mini-apps en HTML/CSS/JS.

## Estructura
- `index.html` — página principal
- `style.css` — estilos
- `script.js` — lógica
- `assets/` — imágenes e íconos

## Cómo ejecutar
- Doble clic en `index.html`, o
- Instala la extensión **Live Server** en VS Code y ejecútalo con recarga en vivo.
